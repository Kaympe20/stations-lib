# `stations-lib`

`stations-lib` is an `npm` library that can be used to get the station code of a given train station. The search algorithm uses fuzzy search.

An example of its usage can be found [here](https://stations-lib-showcase.pages.dev/).

To use it, simply run

```bash
npm install stations-lib
```

<details>
<summary>Development</summary>
To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
</details>

This project was created using `bun init` in bun v1.2.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
